# Hessam Mamagani Portfolio

This repository contains the code for my personal portfolio website, showcasing my projects, skills, and contact information.

## 🚀 Tech Stack

- React 18
- Vite
- Tailwind CSS
- GitHub Pages for hosting

## 🛠️ Setup and Development

1. Clone the repository:
   ```
   git clone https://github.com/hessammamagani/hessam_portfolio.git
   cd hessam_portfolio
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm run dev
   ```

4. Build for production:
   ```
   npm run build
   ```

5. Deploy to GitHub Pages:
   ```
   npm run deploy
   ```

## 📂 Project Structure

- `src/components/` - React components
- `public/` - Static assets
- `src/Portfolio.jsx` - Main portfolio component

## 📝 Personalization

1. Edit `src/Portfolio.jsx` to update projects and personal information
2. Add your profile picture to `public/profile.jpg`
3. Add your CV/resume to `public/cv/Hessam_Mamagani_CV.pdf`
